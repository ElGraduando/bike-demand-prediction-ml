# Bike Demand Prediction using Machine Learning

## Project Overview
This project develops regression models to predict bike rental demand based on environmental and temporal variables. The objective is to support decision-making in sustainable mobility systems.

## Objectives
- Predict bike rental demand using historical data
- Identify key factors influencing demand
- Compare regression models and evaluate performance

## Methodology
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature engineering
- Model training:
  - Polynomial Regression
  - Lasso Regression
- Hyperparameter tuning
- Model evaluation using:
  - RMSE
  - MAE
  - R² score

## Results
The models provided insights into demand behavior and identified key variables influencing bike usage, supporting data-driven mobility planning.

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib / Seaborn

## Repository Structure
- `notebooks/`: analysis and experimentation
- `src/`: modular code
- `data/`: dataset (if included)
